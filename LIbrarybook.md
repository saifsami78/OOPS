## Fields (all private)

- `author` (String)
- `isAvailable` (boolean)

---

##  Constructor
- Takes `title` and `author` as parameters
- `isAvailable` should **automatically start as `true`** (every new book is available)

---

##  Getters & Setters
- Getters for **all 3 fields**
- Setter only for `title` and `author`
- *(availability should only change through borrow/return methods)*

---

##  Methods

### `borrowBook()`
borrows if available, else shows error message

### `returnBook()`
returns if borrowed, else return error message

### `getBookInfo()`
Prints all book details in a nice format.
