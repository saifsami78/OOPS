# 📚 Library Book System

## Problem
Build a LibraryBook class that manages book borrowing.

### Fields
- `title` (String)
- `author` (String)
- `isAvailable` (boolean)

### Requirements
- Constructor sets title & author, availability defaults to true
- borrowBook() — borrows if available, else shows error
- returnBook() — returns if borrowed, else shows error
- getBookInfo() — prints all book details
